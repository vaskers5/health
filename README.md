# Еженедельный ГАЙД (8 недель): зарядка + силовые (Пн/Ср/Сб)

**Календарь:**  
Пн — ДЕНЬ 1 (Верх + кор) • Ср — ДЕНЬ 2 (Низ + кор) • Сб — ДЕНЬ 3 (Спина/плечи/руки + кор).  
Вт/Чт/Пт/Вс — только зарядка (утро/вечер), по желанию лёгкое LISS‑кардио 20–30 мин (вело/эллипс/ходьба).

**Принципы и ограничения (по МРТ):** нейтральный позвоночник; без тяжёлых вертикальных жимов, приседов/становой со штангой, жёстких скручиваний и гиперпрогибов. Темп: подъём без рывка, эксцентрика 2–3 с. Отдых 60–90 с. **RIR 1–2.** Если появляется боль — останавливаешься/меняешь вариант.

---

## ПРОТОКОЛЫ ЗАРЯДКИ (каждый день одинаковые)

### УТРО — 15 минут
1) Дыхание **90/90** — 6–8 вдохов/длинных выдохов.  
2) **Cat–Camel** — 6 плавных циклов без крайних амплитуд.  
3) **Chin‑tucks** — 2×8 (удерж. 3 с).  
4) **Wall‑slides** — 1×10.  
5) **McGill curl‑up** — 3×10‑сек удержаний, чередуя стороны.  
6) **Боковая планка** — 2×20–30 с/сторона (можно с колен).  
7) **Bird‑Dog** — 2×6/сторона (удерж. 5 с).  
8) Опц.: **полувис/вис** 1–2×20 с *или* «**сфинкс**» 3×10 с (мягкая декомпрессия).

### ВЕЧЕР — 15 минут
1) 6 спокойных вдохов/выдохов.  
2) **Open‑Book** — 8/сторона.  
3) **Thoracic extension** на ролике — 6 медленных «прокатов».  
4) **Face‑Pull** резинкой — 2×12.  
5) Растяжка **сгибателей бедра** у стены — 45 с/сторона.  
6) **Фигура‑4** (пириформис) — 45 с/сторона.  
7) **90/90 hip‑switches** — 10 переключений.  
8) **Pallof‑press** 2×10/сторона *или* **Dead‑Bug** 2×8.  
9) Финиш: «поза ребёнка» до **нейтрали** *или* 6 длинных выдохов лёжа.

> ⚠️ Боль ≠ цель. Любое обострение >24–48 ч → уменьшить объём/амплитуду, вернуться постепенно.

## ШПАРГАЛКИ ПО ДНЯМ НЕДЕЛИ

### ПОНЕДЕЛЬНИК — ДЕНЬ 1 (Верх + кор)
- **Суперсет A**:  
  1) **Подтягивания** — 4×6–10  
  2) **Жим гантелей на наклонной 30–45°** — 4×8–12  
- **Суперсет B**:  
  3) **Тяга к груди лёжа / chest‑supported row** — 3×8–12  
  4) **Кроссовер** (угол по неделе) — 3×12–15  
- **Суперсет C**:  
  5) **Face‑Pull** — 3×12–15  
  6) **Pallof‑press** — 3×10/сторона  
- Утро/вечер — по протоколам зарядки.

### ВТОРНИК — День восстановления
- **Утро/вечер** — по протоколам зарядки.  
- По желанию: **LISS‑кардио 20–30 мин** (вело/эллипс/ходьба).

### СРЕДА — ДЕНЬ 2 (Низ + кор)
1) **Leg‑Press** — 4×10–12  
2) **Разгибания ног** — 3×12–15  
3) **Сгибания ног** — 3×10–12  
4) **Hip‑Thrust / Glute‑Bridge** — 3×8–12  
5) **Абдукция** (лента/машина) — 2–3×15  
6) **Икры** — 3×12–15  
7) **Боковая планка** 3×20–30 с/сторона + **Bird‑Dog** 2×6/сторона  
- Утро/вечер — по протоколам зарядки.

### ЧЕТВЕРГ — День восстановления
- **Утро/вечер** — по протоколам.  
- Опц.: прогулка 30–45 мин.

### ПЯТНИЦА — День восстановления
- **Утро/вечер** — по протоколам.  
- Мобилизация грудного отдела 3–5 мин (ролик).

### СУББОТА — ДЕНЬ 3 (Спина/плечи/руки + кор)
- **Суперсет A**:  
  1) **Тяга горизонтального блока** (лучше с грудным упором) — 4×8–12  
  2) **Жим «лэндмайн» под ~45°** *или* жим гантелей сидя со спинкой — 4×8–12  
- **Суперсет B**:  
  3) **Подтягивания** узким/нейтральным — 3×6–10  
  4) **Разведения в стороны** — 3×12–15  
- **Суперсет C (руки)**:  
  5) **Сгибания на инклайн‑скамье 45°** — 3×10–12  
  6) **Трицепс канатом (вниз/над головой)** — 3×12–15  
- **Core‑финал**: **Планка** 3×30–45 с + **Pallof** 2×10/сторона  
- Утро/вечер — по протоколам зарядки.

### ВОСКРЕСЕНЬЕ — День восстановления
- **Утро/вечер** — по протоколам.  
- Лёгкая активность: прогулка/мобилизация 20–30 мин.

## РОТАЦИЯ ВАРИАЦИЙ ПО НЕДЕЛЯМ (1–8)

**Нед. 1–2 — База (техника/объём)**  
- **Пн (Д1):** Н1 — подтягивания *нейтральные*; жим гантелей 30–35°; тяга chest‑supported (DB); кроссовер сверху‑вниз.  
  Н2 — подтягивания *обратный хват* (медленная эксцентрика); жим гантелей 30–35°; тяга Т‑грифа с упором; кроссовер снизу‑вверх.  
- **Ср (Д2):** Leg‑Press средняя стойка; разгибания/сгибания; Hip‑Thrust; абдукция; икры.  
- **Сб (Д3):** Тяга гориз. V‑ручка; жим лэндмайн 2‑ручно; подтягивания узкие; разводки; инклайн‑бицепс; трицепс вниз.

**Нед. 3–4 — Плотность (чуть короче отдых, больше пампа)**  
- **Пн:** Н3 — подтягивания с эксцентр. 3–4 с; жим на наклонной **в Смите** 20–25°; тяга гориз. **с грудным упором**; кроссовер средняя траектория.  
  Н4 — лат‑тяга широким; остальное как Н3.  
- **Ср:** Leg‑Press низкая/узкая постановка (акцент квадры); разгибания с паузой; сгибания с медленной эксцентрикой; Hip‑Thrust с паузой 2 с.  
- **Сб:** Тяга гориз. *обратный хват*; жим лэндмайн **1‑ручно** (split‑stance); подтягивания — негативы 3–4 с; разводки **сидя с упором**; трицепс **над головой**.

**Нед. 5–6 — Интенсификация (безопасно)**  
- **Пн:** Подтягивания с поясом (если 8–10 чистых) — **топ‑сет 6–8**, затем 2 бэкофф‑сета 8–12; жим гантелей **топ‑сет + бэкофф**; тяга chest‑supported **топ‑сет + бэкофф**.  
- **Ср:** Leg‑Press **топ‑сет + бэкофф**; сгибания — **топ‑сет + бэкофф**; Hip‑Thrust **топ‑сет + бэкофф**.  
- **Сб:** Тяга гориз. **топ‑сет + бэкофф**; жим гантелей сидя со спинкой **топ‑сет + бэкофф**; подтягивания — с поясом или обычные 6–10.

**Нед. 7 — Памп‑акцент**  
- **Пн:** Подтягивания — кластер 3+3+3 (20 с паузы) *или* медленная эксцентрика 4 с; жим гантелей 12–15 + **1 дроп‑сет** в последнем; тяга chest‑supported 12–15.  
- **Ср:** Leg‑Press 15–20; разгибания/сгибания — **myo‑reps**; Hip‑Thrust 12–15 + дроп‑сет.  
- **Сб:** Тяга гориз. 12–15; жим в машине/лэндмайн 12–15; разводки 15–20; руки 12–15 (+/‑ дроп‑сет в последнем).

**Нед. 8 — Дeлод**  
- **Все дни:** убрать интенсификаторы, **−35–40 %** подходов, вес −10–15 %. Пример: вместо 4× → 2×, вместо 3× → 2×. Держи технику, уходи с ощущением «легко».

> *Питание/сон:* белок 1.6–2.2 г/кг, дефицит/поддержание на сушке; сон 7–8 ч. После сушки для набора — +200–300 ккал, +1 подход в базовых, отдых 90–120 с, тот же сплит.
